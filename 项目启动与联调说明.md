# 项目启动与前后端联调说明

## 一、后端启动（Spring Boot）
1. 进入 backend 目录：
   ```sh
   cd backend
   ```
2. 构建并启动后端服务：
   ```sh
   mvn clean package
   mvn spring-boot:run
   ```
3. 默认服务端口为 8080，接口基础地址为：
   - http://localhost:8080/api/...

## 二、前端启动（Vue + Vite）
1. 进入 frontend 目录：
   ```sh
   cd ../frontend
   ```
2. 安装依赖（首次启动需执行）：
   ```sh
   npm install
   ```
3. 启动前端开发服务器：
   ```sh
   npm run dev
   ```
4. 默认前端端口为 5173，访问地址：
   - http://localhost:5173

## 三、前后端联调说明
- 前端通过 Axios 调用后端接口，需保证 API 地址指向后端（如 http://localhost:8080）。
- 如有跨域问题，后端已支持 CORS，前端无需额外配置。
- 可在 .env 或 src/utils/request.js 等文件中配置后端 API 基础地址。
- 启动后可直接访问前端页面，进行各业务功能和 AI 对话测试。

---
如遇端口冲突或访问异常，请检查端口占用或防火墙设置。
